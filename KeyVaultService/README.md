SQL:
public_keys - публичные ключи для кажддого пользователя. При помощи них 
шифруються данные отправленные ему.

user_backup - бэкапы ключей

send_new_key -  случае новой сессии или нового приватного ключа, он шифруеться старым приватным ключем 
и сохраняеться в бд, что бы другие устройства так же смогли получить новый ключ.
